# 📝 Task Manager App (Full Stack MERN Project)

A full-stack Task Manager application built using **React, Node.js, Express, and MongoDB**.  
It includes **JWT authentication, CRUD operations, and protected routes**.

---

## 🚀 Features

### 👤 Authentication
- User Registration
- User Login
- JWT Token Authentication
- Protected API Routes

### ✅ Task Management
- Create Tasks
- Read Tasks
- Update Tasks
- Delete Tasks
- Mark tasks as completed

### 🔐 Security
- Password hashing using bcrypt
- JWT-based route protection
- Middleware authentication

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
